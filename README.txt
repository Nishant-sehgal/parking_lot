Parking Lot application written in Java.

It uses maven for building the project.
can run mvn clean install if want to build a jar.

Go to the project home directory i.e parking_lot and run sh bin/parking_lot

Steps
	1. cd parking_lot
	2. sh bin/parking_lot

Running test cases from file.
Steps:
	1. cd parking_lot (project root directory)
	2. java -jar target/parking_lot-0.0.1.jar testcase.txt
